title: "How to connect ChatGPT to Openwork"
description: "How to use ChatGPT OAuth with Openwork"
---

Connecting ChatGPT with openwork takes four clicks.

- Go to `Settings` \> `Connect Provider` \> `OpenAI` \> `ChatGPT Pro/Plus`

<Frame>
  ![Settings page showing Connect Provider button](/images/chatgpt-connect-provider-settings.png)
</Frame>

Then, login into `openai Auth`  with a valid subscription email

<Frame>
  ![OpenAI authentication login page](/images/chatgpt-openai-auth-login.png)
</Frame>

Once you have finished the auth, you can change the model below in the chat.

<Frame>
  ![Model selector dropdown in chat](/images/chatgpt-model-selector.png)
</Frame>

You will be able to choose among any model offered by all the providers you're connected to:

<Frame>
  ![Model list showing available models from all providers](/images/chatgpt-model-list.png)
</Frame>
